# 227화 구조 신호의 값

7구역 외곽 플랫폼은 폐허처럼 보였다.
기둥은 녹슬었고, 표지등은 꺼져 있었다.
하지만 열감지 화면에는 사람 체온이 빽빽했다.

“폐허 코스프레.”
민우가 중얼거렸다.
“사람은 안에 바글바글.”

세린 팀은 점검 통로로 진입했다.
태수가 원격으로 잠금 코드를 풀고,
나래가 카메라 사각을 실시간으로 표시했다.

1.

첫 격실에서 들린 건 비명보다 조용한 기침이었다.
쇠창살 안에 열세 명.
연령과 성별이 제각각.
팔목에는 같은 코드 문신.

선아가 숨을 죽였다.
“생체 담보 대기실…”

세린이 문을 열려다 멈췄다.
태수가 무전으로 막았다.
“지금 전부 풀면 동선이 터진다.
코어 먼저.”

민우가 이를 악물었다.
“또 순서 문제냐.”

세린은 격실 안 사람들과 눈이 마주쳤다.
어떤 소년이 입모양으로 말했다.

_살려줘._

세린이 짧게 답했다.
“다시 올게.
이번엔 진짜로.”

2.

코어로 향하는 복도 중간,
나래가 비상 계획을 꺼냈다.

“자동 개방 예약 걸어.
우리가 코어 장악하면 3분 뒤 격실 동시 오픈.”

태수가 즉시 동의했다.
“좋아. 구조를 지연하지 말고, 동기화하자.”

작은 보상이었다.
이번엔 누구도 완전히 버리지 않았다.

세린이 키패드에 코드를 입력했다.

[격실 1~4 자동 개방 예약]
[트리거: CORE_ACCESS TRUE]

선아가 고개를 끄덕였다.
“기록했다.”

3.

문제는 다음 순간 터졌다.
예약 신호가 내부망을 타고 올라가며,
관리자 패널에 자동 알림이 뜬 것이다.

[비인가 구조 시퀀스 감지]
[위치: 점검동선 C-12]

나래가 욕을 삼켰다.
“젠장, 알림 우회가 안 됐어.
우리 좌표 다 찍혔다.”

복도 끝 방탄 셔터가 내려오고,
천장 포트에서 드론 네 기가 튀어나왔다.

민우가 앞에 섰다.
“뒤는 내가 막아.
너희 셔터 코드 열어!”

총성과 금속음이 연달아 터졌다.
태수는 피를 흘리면서도 키패드를 두드렸다.
세린은 셔터 틈에 손을 넣고 끝까지 밀어 올렸다.

그 순간,
격실 채널에서 사람들 함성이 들렸다.

문이 열린 것이다.

보상은 도착했고,
반전은 더 큰 전투를 불렀다.

셔터 너머 중앙 코어 표지판이 빛났다.
