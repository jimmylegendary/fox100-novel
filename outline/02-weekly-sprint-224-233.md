# 주간 10화 딥 스프린트 아크 (224~233)

## 코어 축
- **주 갈등(Conflict):** FOX-MARKET의 잔여 공급자가 동해 7구역 해상 플랫폼에서 재개장을 준비한다. 세린 팀은 숨겨진 음성 트랙(HJ-Voice)을 해독해 선제 타격해야 한다.
- **주 보상(Reward):** 생체담보 체계를 완전히 끊을 수 있는 ‘원본 키(Origin Key)’ 위치와 작동 규칙 확보.
- **주 반전(Reversal):** 원본 키는 파괴용이 아니라 ‘인수용’이며, 세린 본인 혈통/권한이 최종 트리거로 연결되어 있다.

## 10화 케이던스 (Conflict / Reward / Reversal)
1. **224화**: 히든 트랙 해독 시작, 동해 7구역 좌표 확인 / 팀 재정비 / 트랙 끝에 ‘탑승자 명단’ 반전
2. **225화**: 항만 잠입 루트 확보 갈등 / 내부 협력자 접선 성공 / 협력자가 사실 이중 브로커였음
3. **226화**: 해상 이동 중 추격전 / 암호 키 1차 조각 확보 / 조각이 ‘세린 권한 인증’을 요구
4. **227화**: 7구역 외곽 플랫폼 침투 / 감금 피해자 일부 구조 / 구조 신호가 역으로 위치 노출
5. **228화**: 중앙 코어실 접근전 / 생체담보 로그 원본 획득 / 로그에 한서진의 미공개 서명 발견
6. **229화**: 한서진 서명 진위 공방 / 진짜 의도(시장 내부 파괴용 백도어) 확인 / 백도어가 반쯤 봉인됨
7. **230화**: 봉인 해제 위해 세린 단독 링크 결단 / 플랫폼 제어권 일부 획득 / 제어권 행사 시 세린 기억 손실 리스크 발생
8. **231화**: 팀 내 찬반 분열 / 대체 알고리즘 완성으로 리스크 완화 / 대체안 실행 직전 외부 무력 개입
9. **232화**: 무력 개입 속 다중 전선 / 코어 인수 성공 직전 / ‘원본 키는 폐기가 아닌 승계’라는 핵심 반전 확정
10. **233화**: 승계할지 파기할지 최종 선택 / 세린, 조건부 승계(피해자 협의체 공동키) 선언 / 바다 아래 2차 시장(심해 11구역) 떡밥 오픈

## 실행 원칙
- 매 화 말미 클리프 유지
- 갈등→작은 보상→즉시 반전의 리듬 고정
- 기술/규칙 설명은 대사+행동으로 분산
